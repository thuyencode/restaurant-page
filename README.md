## Getting started

First, install all the dependencies:

```bash
npm i
# or if you're using Bun
bun i
```

To run the dev server:

```bash
npm run start
# or if you're using Bun
bun start
```

To export the production build:

```bash
npm run build

# or if you're using Bun
bun run build
```
